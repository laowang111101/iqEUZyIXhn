# 蛋糕商城销售管理系统 java1198

## 项目概述

本项目是一款基于SSM（Spring、SpringMVC、MyBatis）框架的蛋糕商城销售管理系统。该系统集成了购物车、商品分类、前台下单、后台管理等核心功能模块，并提供完整的源码、安装视频、所需环境及数据库数据，同时配有详尽的文档资料。

## 技术栈

- **后端：** Java、Spring、SpringMVC、MyBatis
- **前端：** JSP、jQuery

## 功能模块

- **用户端：**
  - 购物车
  - 商品分类
  - 前台下单
- **管理员端：**
  - 后台管理

## 系统角色

- 用户：浏览商品、添加购物车、下单等操作。
- 管理员：负责商品的分类管理、订单管理、用户管理等功能。

## 运行环境

- **开发工具：**
  - IntelliJ IDEA（年份不限）
- **数据库：**
  - MySQL 5.1 及以上（含 8、9 版本）
- **Java环境：**
  - JDK 1.8 及以上（17以下）

## 部署步骤

1. 准备开发环境：IDEA、MySQL、JDK
2. 创建数据库，导入数据库脚本
3. 部署应用到服务器
4. 运行项目，进行测试

## 核心亮点

- **双端登录：** 用户端与管理员端分离，确保系统使用的灵活性与安全性。
- **无需手动填充数据：** 提供完整的数据库数据，便于用户快速启动项目。
